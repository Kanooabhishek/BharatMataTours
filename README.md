# Bharat Mata Tours

Prototype website for Bharat Mata Tours: a boutique India and South Asia travel-planning brand for Dutch and Dutch-speaking Belgian travelers.

## Project Type

This is a static website prototype built with:

- `index.html`
- `styles.css`
- `app.js`
- local optimized image assets in `assets/`

## Deployment

The current Vercel project is `bharat-mata-tours`.

Live deployment:

```text
https://bharat-mata-tours.vercel.app
```

The raw image sourcing folder is intentionally excluded from Git and Vercel deployment. Website-ready optimized images are stored in `assets/images/`.
